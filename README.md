## 🚀 功能特点

- ✅ **完全离线运行**：无需网络，一次导入永久使用
- ✅ **多套试卷支持**：可同时导入几十套完整试卷
- ✅ **三种练习模式**：顺序练习 / 随机练习 / 错题本强化
- ✅ **自动保存进度**：答案、错题本地存储，下次打开继续
- ✅ **交卷评分系统**：实时统计正确率，展示每道题对错
- ✅ **全屏沉浸模式**：鸿蒙原生全屏，无状态栏干扰
- ✅ **一键退出提示**：“再按一次退出应用” 防止误退
- ✅ **开源免费**：所有代码公开，可自行修改定制
